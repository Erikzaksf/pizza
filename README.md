# Pizza order/calculator

This is a solo programming project for Epicodus.
It will give the user a choice of pizza size and toppings and display the total cost once submitted.


### Prerequisites

An up to date web browser.
Ability to clone the repository.
Browser with ability to keep local storage.


### Installing

Clone the repository on your local system and run the index.html file contained in the root folder.


## Deployment

Ready to upload to a server/domain, or locally on a browser.

## Specs


* 1. Return the value of size selected by user.

  input: large $20
  outout: $20

* 2. Return sum of toppings selected.

  input: pepperoni $2, onions $1
  output: $3

* 3. Return total cost of crust and toppings.

  input: large $20, pepperoni $2, onions $1
  output: $23
